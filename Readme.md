# Bricked

Tetris++ on iOS and iPadOS. 

- New blocks
- 12x24 play area
- Much faster pace
- Gravity for falling blocks
- Classic Mode, Arcade Mode, Endless Mode

## Cloning and Running Project


```bash
git clone https://github.com/willieLjohnson/bricked.git
cd bricked

pod install
open Bricked.xcworkspace
```
